# Pet Care Scheduler

The *Pet Care Scheduler* is a web-based system designed to assist pet owners, veterinarians, and pet clinics in managing pet-related activities efficiently.  
The system provides tools for scheduling veterinary appointments, setting automated reminders for pet care tasks, and storing and tracking pet health records.

By digitizing the pet care process, the system promotes responsible pet ownership and ensures that pets receive timely medical attention and consistent care.

---

## Tech Stack

- *React* – For building dynamic and responsive UI components  
- *Tailwind CSS* – For fast, modern, and responsive styling  
- *Supabase* – For backend services (authentication, database, and storage)  
- *Git + GitHub* – For version control and collaboration
